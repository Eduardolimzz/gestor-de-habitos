# Gestor de Hábitos Saudáveis

Projeto desenvolvido para a disciplina de Projeto Integrador (PI), com o objetivo de aplicar conceitos de desenvolvimento mobile e backend utilizando React Native e Node.js.

## Descrição

O sistema consiste em um aplicativo mobile para gerenciamento de hábitos saudáveis, permitindo ao usuário cadastrar, visualizar e acompanhar suas atividades diárias.

A aplicação é composta por duas partes:

- Frontend mobile desenvolvido em React Native
- Backend desenvolvido em Node.js com Express

## Objetivo

Desenvolver uma aplicação completa (frontend + backend) aplicando conceitos de:

- Componentização
- Navegação entre telas
- Consumo de API
- Estruturação de código
- Boas práticas de desenvolvimento

## Tecnologias Utilizadas

### Mobile
- React Native (Expo)
- JavaScript
- Axios

### Backend
- Node.js
- Express

## Estrutura do Projeto
```
gestor-de-habitos/
├── backend/ # API
├── mobile/ # Aplicação mobile
```

## Como Executar o Projeto

### Backend

```bash
cd backend
npm install
npm run dev
```

### Mobile
```bash
cd mobile
npm install
npx expo start
```
